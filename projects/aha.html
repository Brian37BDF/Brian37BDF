Currently redesigning a realty group’s website and home search functionality.
About this project:

American Home Agents, a realtor group based in Aurora, CO was looking for a fresh design from it’s current Wordpress template-based site. I worked with their team to come up with some different color palettes, as well as a couple different directions for the new site. Below are some of the different hero sections I pitched for the homepage. I explored some different interaction patterns to find which search functionality would be most useful for a home shopper. Because the project is still in progress, a lot of the content is still block-framed.

Home Listings:

Another goal of the project was to rework the home search functionality of the listings page. Previously American Home Agents was relying on external service to provide the listings, but I figured they could utilize a refined way to sort and fine-tune parameters while searching. With this focus in mind, I came up with the filter system you can see below. The user can add different parameters like location, price, square footage, etc. and watch as the results are updated.
