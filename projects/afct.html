Intranet Redesign:

Before I began at AFC, the old intranet was a pretty basic HTML page with links to some important documents, payroll and CRM services, and email links for HR and tech support. My task was to look at all of the current uses of the intranet, make those more accessible and easy to find, while also coming up with additional ways to increase efficiency. This included gamified leaderboards for the sales team to compare successes, a calendar and announcement area for HR to communicate to the entire team, as well as a tool section for employees to navigate and find the applets needed to complete a task. Lastly I wanted a cohesive and clean look for this dashboard to match a recent homepage redesign, complete with a fresh name and logo!

Landing Page Style Guide:

I am especially proud of this particular project as it was an idea that I came up with that I knew would make the design team happy. As a smaller design team, I often put on my front-end developer hat to work on the 100+ landing pages we had created over the years for various marketing campaigns. As these often shared certain details, I realized I was repeating work when I had to configure similar features across multiple pages. I realized I could create a standardized style across all future landing pages, so that new pages could be created simply using the provided templates I created. This solved an earlier problem where I felt as if I was the only designer with the “keys” to the landing pages, as I was the only one working in this repository. This system opened things up to where the other designers could create pages using the responsive elements I had already developed. A bonus functionality of this tool was that it also made it easier to inject these HTML/CSS elements into Optimizely, our A/B testing platform so we could rapidly improve our pages through experimentation. I called the project 'Lander', and even made a little bit of branding, too! CRUD Interface for Home Page:

The AFC homepage features a lot of useful content across our Learning Center, so we needed a tool that our marketing team could use to add or edit anything across the site. This meant creating an article editor, complete with abilities to configure the URL, SEO, and other meta data. I also created a browser in which to see what content had already been created or updated, and to allow the team the ability to schedule future posts.
